CoinPay — современное мобильное приложение для управления финансами, реализованное на Flutter с использованием Clean Architecture,
go_router для навигации, 
flutter_bloc (Cubit) для управления состоянием,
Dio для сетевых запросов и Hive для локального хранения данных.
